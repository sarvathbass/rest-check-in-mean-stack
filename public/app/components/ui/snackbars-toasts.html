Snackbars & Toasts

Toasts

Toasts provide an easy way for users to receive unobtrusive alerts. Toasts are also placed and sized responsively.
Test it out by clicking the button below on different devices.

Toast!

To do this, call the Materialize.toast() function programatically in JavaScript.

Code Sample
// Materialize.toast(message, displayLength, className, completeCallback);
Materialize.toast('I am a toast!', 4000) // 4000 is the duration of the toast

Another way to add this into your application is to add this as an onclick event to a button.

Code Sample
<a class="btn" onclick="Materialize.toast('I am a toast', 4000)">Toast!</a>
Custom HTML

You can pass in an HTML String as the first argument as well. Take a look at the example below, where we pass in text as well as a flat button. If you call an external function instead of in-line JavaScript, you will not need to escape quotation marks.

Custom HTML toast
Code Sample
var $toastContent = $('<span>I am custom HTML toast content</span>');
Materialize.toast($toastContent, 5000);
Callback

You can have the toast callback a function when it has been dismissed.

Callback Toast!
Code Sample
<a class="btn" onclick="Materialize.toast('I am a toast', 4000,'',function(){alert('Your toast was dismissed')})">Toast!</a>
Styling Toasts

Your toasts can be easily customized. You can pass in classes as an optional parameter into the toast function. The rounded class is added by default for you, but you can create your own CSS classes and apply them to toasts. Checkout out the full example below.

Round Toast!
Code Sample
Materialize.toast('I am a toast!', 3000, 'rounded') // 'rounded' is the class I'm applying to the toast
